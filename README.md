# Shadcn UI 测试项目

这是一个使用 Next.js 14 和 Shadcn UI 构建的测试项目，用于展示各种 UI 组件和主题系统。

## 技术栈

- Next.js 14
- TypeScript
- Tailwind CSS
- Shadcn UI
- Radix UI

## 开始使用

### 本地开发

```bash
git clone https://github.com/nobruf/shadcn-landing-page.git
```

2. Go into project

```bash
cd shadcn-landing-page
```

3. Install dependencies

```bash
npm install
```

4. Run project

```bash
npm run dev
```
